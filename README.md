## Install dependencies

```bash
bun install
```

## Quickstart
```bash
bun build index.ts --compile --outfile=chat
./chat
```


## Development
```bash
bun run index.ts 
```

### Environment variables
```
OPENAI_API_KEY (REQUIRED)
OPENAI_MODEL
SYSTEM_PROMPT
```
